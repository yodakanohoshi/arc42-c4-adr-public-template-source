## 正確な契約を別仕様へ分けるもの

- HTTP API: OpenAPIなどのAPI記述
- Event / Message: AsyncAPIまたはイベントカタログ
- File連携: ファイルレイアウト、文字コード、再送規則
- Database共有が避けられない場合: 所有権、Schema、変更手順、アクセス制御
