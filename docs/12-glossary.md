# 12. 用語集

| 用語 | 定義 |
|---|---|
| 対象システム | このアーキテクチャ文書が説明するシステム |
| 外部システム | 対象システムの境界外にあり、連携するシステム |
| Building Block | arc42におけるシステム構成要素 |
| Container | C4におけるアプリケーションやデータストアなどの実行単位 |
| Component | Container内部の主要な責務単位 |
| Runtime Node | ソフトウェアを配置・実行する論理または物理ノード |
| System Context | 対象システム、利用者、外部システムの関係を示すビュー |
| Interface Contract | システム間の入出力、振る舞い、互換性の合意 |
| ADR | Architecture Decision Record。重要な設計判断と理由の記録 |
| SLI | 品質を測定する指標 |
| SLO | SLIに対して定める目標水準 |
| RTO | 障害後にサービスを復旧する目標時間 |
| RPO | 障害時に許容するデータ損失時点の目標 |
| Runbook | 定型的な運用・障害対応手順 |
| 正の情報源 | ある情報について正式かつ優先して参照する場所 |
